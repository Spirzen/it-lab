## Лаборатория

<DocCardList />

**Лаборатория** — это практическое пространство, в котором теоретические знания переводятся в действие. Раздел включает структурированные материалы для освоения, закрепления и верификации компетенций: от базовых упражнений до комплексных профессиональных сценариев.  

Содержание раздела охватывает следующие категории:  
- **Вопросы и опросы** — инструменты самодиагностики, включая чек-листы и квизы для оценки понимания тем;  
- **Задачи** — упражнения, ранжированные по уровням сложности (начальный, продвинутый, экспертный);  
- **Кейсы** — разборы реальных или смоделированных профессиональных ситуаций с анализом решений;  
- **Планы развития** — рекомендованные траектории обучения для различных IT-профессий, а также для детей и начинающих;  
- **Примеры кода и решений** — иллюстрации корректных подходов к реализации типовых и нетиповых задач;  
- **Тренажёры** — подборка онлайн- и офлайн-инструментов для интерактивной практики;  
- **Экзамены** — комплекты заданий, подтверждающие уровень освоения дисциплины или профессии;  
- **Эксперименты** — документированный опыт тестирования нестандартных подходов, технологий или методик.  

Материалы Лаборатории могут использоваться как для автономного обучения, так и в рамках образовательных программ, обеспечивая воспроизводимость, измеримость и адаптивность учебного процесса. Теория по темам — в [Энциклопедии](/encyclopedia/intro); блок **«Связанные темы»** внизу каждой статьи ведёт к соответствующим разделам. Для HTTP- и API-проверок в практикумах держите под рукой [утилиту curl](/encyclopedia/2-system-network/2-05-terminal/1133) и [галерею примеров curl / fetch](/lab/Примеры/1133). Для Python GUI — [Tkinter — окна и виджеты](/lab/Примеры/1124) (теория — [Tkinter и GUI](/encyclopedia/5-languages/5-02-python/311)); для C# desktop — [WinForms и WPF — простые окна](/lab/Примеры/1138) (теория — [Windows Forms (WinForms)](/encyclopedia/4-code-dev/4-11-desktopnye-prilozheniya/115), [Первая форма WPF — XAML, стили и шаблоны](/encyclopedia/4-code-dev/4-11-desktopnye-prilozheniya/119)); для рисования — [Turtle](/lab/Примеры/111) (Python) и [p5.js](/lab/Примеры/1114) (браузер). Для CI/CD — [GitHub Actions — рецепты](/lab/Примеры/1134) (теория — [8.04 DevOps](/encyclopedia/8-infra-security/8-04-devops-ci-cd/2112)). Для Java — [консольные задачи](/lab/Примеры/1131) и [Swing — окна и кнопки](/lab/Примеры/1143) (теория — [раздел Java](/encyclopedia/5-languages/5-03-java/intro), GUI — [JavaFX и GUI](/encyclopedia/5-languages/5-03-java/311); [популярные запросы](/lab/Примеры/1131#8-популярные-запросы-из-google) — сумма массива, bubble sort, reverse string). Для React — [компоненты-рецепты](/lab/Примеры/1146) (теория — [React - библиотека для пользовательских интерфейсов](/encyclopedia/5-languages/5-01-javascript/27), старт — [272](/encyclopedia/5-languages/5-01-javascript/272)).
